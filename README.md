Resumo da Aula – Fundamentos de JavaScript 19/03/2026

Nesta aula foram apresentados conceitos básicos de JavaScript importantes para o desenvolvimento web.

Foram abordados os tipos primitivos, como boolean e number, e o uso do typeof para identificar o tipo de uma variável. Também foi explicado o uso de variáveis com var e const, além da diferença entre escopo global e local dentro de funções.

A aula mostrou como criar funções simples e como funcionam as comparações em JavaScript, destacando a diferença entre == e ===.

Por fim, foram apresentados operadores aritméticos, relacionais e lógicos, que são essenciais para realizar cálculos e criar condições no código.
